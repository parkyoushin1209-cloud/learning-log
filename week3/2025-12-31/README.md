운영체제: 응용 프로그램에 필요한 자원(시스템 자원- cpu, 메모리, 보조기억장치, 입출력 장치 등) 할당 및 프로그램이 올바르게 실행되도록 돕는 프로그램
-> 하드웨어 조작/관리, 문제 해결 실마리 제공

커널: 운영체제의 핵심 서비스를 담당하는 부분

운영체제의 기능
-커널을 포함하지 않는 서비스 => 사용자 인터페이스(컴퓨터와 상호작용할 수 있는 통로)-그래픽 기반 인터페이스(GUI)와 명령어 기반 인터페이스인 CLI가 있음
-응용 프로그램의 하드웨어 자원 직접 접근을 방지함 시스템 호출을 통해서만 접근 가능/시스템 호출 -> 코드 실행 하드웨어 요소 접근-> 다시 사용자모드로
이중 모드: CPU가 명령어를 실행하는 모드를 크게 사용자 모드(운영체제 서비스를 받을 수 없는 모드), 커널 모드로 구분하는 방식

핵심 서비스 -> 프로세서 관리(cpu에서는 한 번에 하나의 프로세서만 실행가능 -> 프로세서 전환 반복[CPU스케줄링]), 파일 시스템 관리, 자원 접근 및 할당

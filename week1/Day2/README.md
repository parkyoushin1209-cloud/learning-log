-프로그래밍
여러가지 정렬 방법 기수정렬, 병합정렬, 버블 정렬 timsort


느낀점:
정렬 알고리즘을 단순히 코드로 외우는 게 아니라 추가적으로 조사해보면서 정렬에서의 안정성이 실무 환경에서 중요하다는 점을 깨닫게 되었다. 이에 관해
timsort가 단순히 이론적으로 만들어진 정렬이 아니라 현실 데이터를 고려해 만들어졌다는 점이 매우 인상 깊게 느껴졌다.

공부를 하다보니 질문이 자연스럽게 떠오르고 이에 대해 더 깊이 파고들어가게 됨으로써 공부의 방향성을 잘 잡아가고 있다고 생각했다.

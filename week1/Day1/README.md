-프로그래밍
기본 연산자의 기능, 논리 연산자를 이용한 관련 식
정렬 방법 학습: 퀵 정렬, 삽입 정렬, 선택 정렬, 계수 정렬

-느낀점
1. 정렬 학습 1회차에서는 코드 구현 자체에만 집중했으나, 이번에는 각 정렬 방식의 장단점과 어떤 경우에 시간이 오래 걸리는지, 메모리 사용 특성은 어떤지까지 함께 고려하며 정렬 알고리즘을 더 깊이 이해할 수 있었다.
2. 비트 관련 연산자를 배우고, 이를 활용한 식들을 직접 구현해보면서 단순 암기가 아닌 연산의 원리를 이해하는 시간을 가질 수 있었다.
3. 1학년 때는 왜 이 과정을 배우는지 체감하지 못했지만, 이번 학습을 통해 이후 학년에서 배우게 될 과목들과의 연관성을 생각해 볼 수 있었고, 전공 과목의 중요성을 다시 느끼게 되었다.
